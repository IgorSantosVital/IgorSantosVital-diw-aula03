Descrevendo algumas tags HTML de formatação de texto

Tag <em>

Esta tag é utilizada para dar ênfase ao texto contido em uma página HTML. O seu comportamento padrão exibe o texto em itálico.

Tag <Strong>

Esta tag é utilizada para dar importância ao texto. O seu comportamento padrão exibe o texto em negrito.

Tag <sup> e <sub>

Estas tags são utilizadas para definir o texto sobrescrito e subscrito respectivamente. São muito utilizados em fórmulas químicas (H2SO4) e expressões matemáticas com potência (23=8), por exemplo.

Tag <ins> e <del>

Estas tags são utilizadas para definir inserções de texto(com a tag <ins>) e remoções de texto (com a tag <del>). Inserções e remoções são comuns em artigos de blogs, quando o autor quer deixar o usuário ciente de que houve uma atualizarão atualização no texto, seja ela uma correção, inserção ou inclusão de palavra(s). O comportamento padrão deixa o texto sublinhado para inserções e tachado para exclusões.

Tag <mark>

Esta tag é utilizada para marcar trechos do texto. É possível marcar tanto palavras específicas quanto um conjunto de palavras. O comportamento padrão deixa o texto na cor preta e com o fundo amarelo.
